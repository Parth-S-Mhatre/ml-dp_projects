# 🎓 Student Performance Dashboard

An AI-powered Streamlit application that predicts student academic performance and provides personalized insights, analytics, and recommendations to help students improve their learning outcomes.

---

## 📌 Project Overview

Student performance is influenced by multiple academic, personal, and environmental factors.

This project uses Machine Learning to analyze these factors and predict a student's academic performance while providing actionable recommendations based on the prediction.

The application is designed for educational purposes and demonstrates how data-driven insights can support academic success.

---

## 🎯 Objectives

* Predict student academic performance
* Analyze key factors affecting learning outcomes
* Visualize performance indicators
* Provide personalized recommendations
* Build an interactive machine learning dashboard

---

## 🌟 Key Features

### 📊 Performance Prediction

Predicts student performance based on multiple inputs including:

* Study Hours
* Attendance
* Sleep Duration
* Family Support
* School Environment
* Academic History

---

### 📈 Interactive Analytics

Visual dashboards help users understand:

* Performance trends
* Academic strengths
* Areas for improvement
* Impact of lifestyle factors

---

### 🎯 Personalized Recommendations

Based on prediction results, the system provides:

* Study recommendations
* Attendance guidance
* Time management suggestions
* Academic improvement strategies

---

### ⚡ Real-Time Predictions

Users can modify inputs and instantly receive updated predictions and insights.
